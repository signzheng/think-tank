# 理性决策的指南针:**数学期望**

如果说逻辑是思维的骨架，那么**数学期望（Mathematical Expectation）**就是理性决策的指南针。

通俗地说，数学期望不是在预测“下一次会发生什么”，而是在计算**“如果这事重复做一万次，平均下来的一次的结局是什么”。**

它是**“上帝视角的平均值”**。

------

### 一、 怎么理解？（从公式到直觉）

用最简单的数学公式表达：

$$期望 (E) = (概率 P_1 \times 结果 V_1) + (概率 P_2 \times 结果 V_2) + ...$$

**直觉案例：抛硬币赌局**

- 规则：抛硬币。正面给你100元，反面你要赔80元。
- 玩不玩？
  - 正面概率 50%，收益 +100。
  - 反面概率 50%，亏损 -80。
  - **数学期望** = $(0.5 \times 100) + (0.5 \times -80) = 50 - 40 = +10$ 元。

结论： 期望是正数（+10元），必须玩！

虽然你下一把可能会输80元，但只要你一直玩下去，平均每一把你都在赚10元。

核心理解误区：

很多人认为期望是“最可能发生的结果”。错！

- 扔一个骰子（1-6点）。期望值是 3.5。
- 但你永远不可能扔出 3.5 点。
- 期望值不是现实中会出现的某个数，它是**所有可能性的重心**。

### 二、 生活中有什么用？（理性人的决策工具）

在生活中，数学期望能帮你穿透表象，看清事情的**“真实价格”**。

#### 1. 识别“智商税”：彩票与赌博

为什么说彩票是穷人的智商税？用期望一算便知。

- 假设一张彩票 2 元。
- 中奖率是 1/1000万，奖金是 500万。
- **这张彩票的真实价值** = $500万 \times (1/1000万) = 0.5$ 元。
- **决策：** 你花 2 元买了一个价值 0.5 元的东西。期望是 **-1.5元**。

**应用：** 凡是期望值为负的游戏（赌场、彩票、资金盘），理性人绝不参与。因为“大数定律”决定了，只要你玩得久，你必输。

#### 2. 职业选择：高风险 vs 铁饭碗

很多人纠结去创业公司还是去大厂，可以用期望来辅助思考。

- **选项A（大厂）：** 100% 拿到 30万年薪。
  - $E(A) = 30万$。
- **选项B（创业）：** 90% 倒闭（年薪0），10% 上市（年薪500万）。
  - $E(B) = (0.9 \times 0) + (0.1 \times 500万) = 50万$。

**决策：** 虽然 $E(B) > E(A)$，也就是创业的平均回报更高。但这里涉及到一个**“效用”**问题（后文会讲）。如果你能承受归零的风险，从数学上看，应该选期望值高的。

#### 3. 恋爱与“沉没成本”

期望是**面向未来**的，而我们往往被**过去**纠缠。

- 现状：你和一个不合适的人谈了3年（沉没成本），很痛苦。
- 纠结：分不分？
- **期望思维：** 不要管过去那3年。计算一下：
  - **不分：** 未来幸福概率 10%。
  - **分：** 虽然痛苦3个月，但未来遇到合适人的概率是 60%。
  - **决策：** 只要 $E(未来) > E(现在)$，立刻止损。

#### 4. 甚至可以用来“犯规”：停车费博弈

这是一个生活中的黑色幽默案例。

- 某路段停车费：20元。
- 贴条罚款：200元。
- 警察巡逻概率：如果警察平均1个月才来查这块区域一次（即概率约为 1/30 ≈ 3%）。
- **逃费的期望成本** = $200 \times 3\% = 6$ 元。
- **决策：** 因为 6元 < 20元，从纯数学角度，**违章停车是“划算”的**。
  - *(注：当然，社会规则和道德成本也是隐形权重，这里仅做数学讨论)*

### 三、 期望的陷阱：为什么有时候不能信期望？

虽然期望是理性的，但作为普通人，完全按期望生活会死得很惨。因为有两个限制：

**1. 爆仓风险（凯利公式的领域）**

- 如果不玩这个游戏，你一分钱没有。
- 如果玩：50%几率赢100亿，50%几率**立刻死刑**。
- **计算：** $E = (0.5 \times 100亿) + (0.5 \times 死) = +50亿$ （假设命有价）。
- 期望收益极高，但你敢玩吗？不敢。
- **道理：** 期望只有在**“可以重复很多次”**时才有效。如果输一次就彻底出局（死亡、破产），那么正期望也没意义。**这也是为什么我们要买保险（虽然保险的期望收益是负的），是为了防止那个不可承受的“归零”。**

\2. 幸存者偏差

期望算的是平均值。但在极端分布（幂律分布）中，平均值会骗人。

- 我和马云的平均资产是几千亿。但这对我没有意义。
- 如果一个事情的成功率只有 0.001%，哪怕回报是无限大，对于只有一次生命的个体来说，期望值再高也大概率是当炮灰。

### 四、 总结

数学期望在生活中的真正用法是：

1. **对于小事、可重复的事：** 严格按照正期望行事。比如投资、时间管理、小额博弈。只要胜率占优，坚持做，时间会给你回报。
2. **对于大事、不可逆的事：** 关注**“最大回撤”**（最坏结果）。如果最坏结果是“死”或“破产”，哪怕期望收益再高，也不要上牌桌。

**理性的人，算计期望；智慧的人，管理风险。**
