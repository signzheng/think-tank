# 易中天：什么是真正的文明？从墨子的“公义”到杨朱的“一毛”

如果说历史是一面镜子，那么先秦诸子的思想就是这面镜子最原本的底色。我们在谈论中华文明时，往往容易陷入宏大的叙事，却忽略了文明最基础的两个维度：**社会的公平与个体的尊严**。

今天，我想聊聊被误解最深的两个人——墨子和杨朱，以及我们身边那些令人啼笑皆非的“现代怪相”。

### 一、 墨子：最早的“机会均等”倡导者

提起墨子，大家只知道“兼爱非攻”，甚至有人拿《红楼梦》里“焦大不爱林妹妹”来抬杠，质问我“爱怎么可能是普遍的？”

这不仅是逻辑问题，更是常识问题。我不爱林妹妹，不代表我不爱张妹妹、李妹妹，更不代表我不能爱父母、爱朋友。所谓的“兼爱”，不是让你谈一场滥情的恋爱，而是一种社会伦理的底线。

墨子对中华文明真正的贡献，其实可以总结为十六个字：

“自食其力，各尽所能，按劳分配，机会均等。”

这十六个字，构成了中国最早、也最朴素的“空想社会主义”蓝图：

1. **反特权**：人人都得劳动，管理国家是劳动，教书也是劳动，绝不允许“不劳而获”。
2. **讲规则**：分配要看贡献大小，而不是看出身贵贱。
3. **重公平**：要想“各尽所能”，前提必须是“机会均等”。能力低的去低岗位，能力高的去高岗位，大家凭本事吃饭。

墨子留给我们的，除了江湖上的“侠义精神”，更是一份关于**社会程序正义**的契约。

### 二、 杨朱：被泼脏水的“人权斗士”

比墨子误解更深的是道家的杨朱。教科书里骂他“自私自利”，因为他说过那句名言：**“拔一毛利天下而不为”**。

乍一听，这人太坏了，拔根汗毛就能造福天下，他居然不干？但大家往往故意忽略了他的后半句：**“悉天下奉一身而不取”**——把全天下的财富给一个人享用，杨朱也坚决不要。

杨朱反对的，从来不是“利天下”，而是**“以利天下的名义，无底线地侵犯个人权利”**。

我们可以做个思想实验。墨子的学生曾去质问杨朱的学生：“拔一根毛就能救天下，干不干？”

这看似是一个道德高地，但逻辑是可以滑坡的：

- 如果拔一根毛可以，那么打你一顿给50块钱，干不干？（干）
- 砍你一条胳膊，拯救一个国家，干不干？（犹豫了，不干）
- 那砍你的头呢？（肯定不干）
- 既然头不能砍、胳膊不能砍、皮肤不能刮，**为什么毛就可以拔？**

杨朱的逻辑振聋发聩：**不要因为个体微小，就可以被随意牺牲。**

只要开了“拔一毛”的口子，权力的手就会伸向你的皮肤、你的肢体，最后是你的头颅。杨朱是一毛不拔，墨子是一毛不留，孟子把他们都骂了。但在今天看来，杨朱其实是在捍卫一条文明的底线：**每个个体都是人，没有一个无辜者应该成为宏大叙事的祭品。**

### 三、 现代怪相：“找领导”与“你幸福吗”

明白了墨子的“公义”和杨朱的“私权”，再看我们当下的社会，就会发现许多荒诞之处。

第一是严重的“官本位”。

有个笑话：喝咖啡发现苍蝇。英国人绅士地走人，日本人愤怒地质问管理，美国人幽默地调侃。而中国人的第一反应是什么？

拍案而起：“把你们领导找来！”

为什么我们习惯找领导？因为在我们的潜意识里，规则不重要，法治不重要，**权力才重要**。这是法治精神的缺失，也是墨子所追求的“机会均等”的反面。

第二是形式主义的“伪关怀”。

央视曾满大街追问：“你幸福吗？”一位农民工兄弟神回复：“我姓曾。”

这是一个极其愚蠢的问题。

幸福是一种纯粹主观的、私人的感受。此刻幸福不代表永远幸福，今天晚上遇到个堵心事，我可能就不幸福了。

我的幸福与否，关你什么事？更何况，当公权力拿着话筒逼问一个标准答案时，得到的只能是“我姓曾”这样软性的消极抵抗。

真正的文明是什么？

真正的文明，是像墨子那样，建立公平的规则，让每个人都能凭本事吃饭；是像杨朱那样，捍卫私人的边界，哪怕是一根汗毛也不容许被神圣化地剥夺。

而不是这年头，如果不装傻充愣，连个领导都当不成。
