# 贝莱德的第四权

这是一篇关于现代资本主义**“影子皇帝”**的深度解剖报告。

你之所以感觉**贝莱德（BlackRock）\**神秘且无处不在，是因为它开创了一种全新的统治模式：它并不直接拥有工厂或军队，它通过\**“管理全世界的钱”\**和\**“控制全世界的风险数据”**，成为了凌驾于许多主权国家之上的**“第四权”**（The Fourth Branch）。

我们可以通过以下四个维度，看懂拉里·芬克（Larry Fink）和他的帝国是如何运作的：

### 1. 体量：富可敌国的“超级房东”

首先，必须建立一个数量级的概念。

贝莱德管理的资产规模（AUM）在高峰期超过**10万亿美元**。

- **这是什么概念？** 这比德国和日本的GDP加起来还要多。如果贝莱德是一个国家，它是仅次于美国和中国的世界第三大经济体。
- **它拥有什么？** 它是几乎所有你叫得上名字的跨国公司的**前三大股东**。
  - 你想喝可乐？它是**可口可乐**的大股东，也是**百事可乐**的大股东。
  - 你想买手机？它是**苹果**的大股东，也是**微软**的大股东。
  - 你想打疫苗？它是**辉瑞**的大股东，也是**强生**的大股东。

这就是所谓的**“共同所有权”**（Common Ownership）。表面上百事和可口在打架，其实背后的老板是同一个。

### 2. 大脑：阿拉丁系统（Aladdin）——金融界的“天网”

如果说钱是贝莱德的肌肉，那么**阿拉丁（Aladdin）**就是它的大脑。这也是贝莱德最可怕、最不为人知的地方。

- **它是什么？** 一个超级风险管理系统（AI+大数据）。
- **它做什么？** 它不仅仅管理贝莱德自己的钱，还出租给其他竞争对手、银行、保险公司甚至**中央银行**使用。
- **恐怖之处：** 据统计，全球有超过**20万亿美元**的资产依赖阿拉丁系统进行风控。这意味着，拉里·芬克掌握着全球金融市场的**“上帝视角”**。他比美联储主席更清楚资金在世界各个角落的流动。
  - *实战案例：* 2008年金融危机和2020年疫情熔断时，**美联储（Fed）**都不知道该救谁、怎么救。他们最后不得不聘请贝莱德，利用阿拉丁系统来指导美国政府救市。**那一刻，贝莱德实际上成为了“央行的央行”。**

### 3. 鞭子：ESG评分——资本的“紧箍咒”

拉里·芬克如何控制那些他并不控股的公司？答案是：**ESG（环境、社会和治理）**。

每年，拉里·芬克都会写一封**《致CEO的信》**。这封信被华尔街视为“圣旨”。

- **逻辑：** 他告诉所有公司：“你们必须关注气候变化，必须保持董事会多元化（多招少数族裔和女性）。如果不听话，贝莱德就会投反对票，或者降低你们的评级，让你们融不到资。”
- **结果：** 这是一个巨大的**权力工具**。通过ESG，贝莱德可以强迫石油公司减产（得罪了红脖子），强迫好莱坞拍“政治正确”的电影（得罪了保守派）。
- 这就是为什么马斯克极其痛恨ESG，称其为“魔鬼的骗局”，因为ESG给了贝莱德干预企业经营的尚方宝剑。

### 4. 旋转门：影子公司与白宫的“同体”

贝莱德不仅控制华尔街，它还直接向华盛顿输送人才，被称为**“新高盛”**。

- **拜登政府：** 拜登上台后，任命布莱恩·迪斯（Brian Deese）为国家经济委员会主任，任命沃利·阿德耶莫（Wally Adeyemo）为财政部副部长。这两人此前都是贝莱德的高管。
- **这种关系是双向的：** 当由于政策变动导致市场波动时，贝莱德总能“恰好”提前布局。

### 总结：它是如何成为“第四权”的？

传统的美国三权分立是：立法、行政、司法。

而贝莱德代表了**“金融/算法权”**。

它不需要通过选举（像总统），也不需要通过考试（像法官）。它利用普通人的养老金（401k）汇聚成巨大的资本池，然后利用这个资本池去**胁迫**企业和政府执行它的意志。

- 对于支持者，它是“负责任的资本主义”，在推动世界向绿色、平等转型。
- 对于反对者（尤其是共和党和红脖子），它是**“觉醒资本主义”（Woke Capitalism）的暴君**，是未经选举就统治世界的“深层政府”代理人。

这也就解释了为什么**德克萨斯州**（共和党大本营）最近宣布要从贝莱德撤资——这是一场**“红州”**对抗**“华尔街皇帝”**的叛乱。

既然聊到了贝莱德利用**ESG**作为武器来推行“政治正确”，这其实是引发美国**“文化内战”**的导火索之一。你有兴趣了解一下这背后的**反击战**吗？比如为什么马斯克和某些州正在发起**“反ESG运动”**？

# 反ESG运动

这就好比《帝国反击战》。

如果说贝莱德通过ESG（环境、社会、治理）评分建立了一种**“金融极权”**，那么现在，马斯克和美国的共和党州正在发起一场激烈的**“反抗军起义”**。

这场反击战极其精彩，因为它不仅是口水仗，而是真金白银的**“资本撤资战”**。

我们可以通过三个关键节点来复盘这场“反ESG运动”：

### 1. 导火索：马斯克的愤怒（“ESG是魔鬼的骗局”）

整件事的高潮发生在2022年5月。当时，标普500 ESG指数（S&P 500 ESG Index）做了一次调整：

- **踢出了特斯拉（Tesla）：** 全球最大的电动车公司，致力于清洁能源，被认为“不环保”。
- **保留了埃克森美孚（ExxonMobil）：** 全球最大的石油巨头，被认为“符合ESG标准”。

**为什么？**

因为ESG不仅仅看**E（环境）**，还看**S（社会）\**和\**G（治理）**。评级机构认为，虽然特斯拉造电车（E好），但马斯克反对工会、公司缺乏多元化高管（S和G差）。而石油公司虽然污染大，但只要它们雇佣足够多的少数族裔高管，写足够漂亮的“多元化报告”，分数就能刷上去。

**马斯克的反击：**

他在推特上暴怒，直接撕下了皇帝的新衣：**“ESG就是一个骗局，是‘社会正义武士’（SJW）伪装成的企业魔鬼。”**

这不仅是他个人的私怨，更是一声集结号，让所有被ESG压迫的保守派意识到：**这个游戏规则从根本上就是针对我们的。**

### 2. 主力军：“红州”的金融反击（德州与佛州的宣战）

马斯克负责在舆论上开炮，而共和党控制的“红州”（Red States）则负责在**资金上断供**。

这就涉及到了美国独特的**“联邦制”**力量。美国各州的**养老金基金（Pension Funds）**拥有数千亿美元的规模，这是华尔街的大金主。

- **德克萨斯州（Texas）打响第一枪：**

  德州是美国的石油心脏。贝莱德通过ESG逼迫石油公司减产、撤资，这直接动了德州的蛋糕。

  于是，德州通过法律，列出了一份**“抵制能源公司黑名单”**。

  - **逻辑：** “贝莱德，既然你抵制我们的石油行业，那我们就认定你‘反德州’。以后德州几百亿的教师退休金、公务员养老金，一分钱都不给你管理。”

- **佛罗里达州（Florida）跟进：**

  州长德桑蒂斯（DeSantis）更加意识形态化。他立法**禁止**该州的养老金在投资时考虑ESG因素。

  - **口号：** “我们只看回报率（Return），不看政治（Politics）。”

**结果：** 这一波“撤资潮”让贝莱德损失了几十亿美元的管理资产。虽然对10万亿的总盘子来说不致死，但**信号极强**——华尔街第一次感到了痛。

### 3. 理论武器：维韦克·拉马斯瓦米与“信托责任”

反击战需要理论指导。这里必须提到一位印度裔神童、现在的共和党新星——**维韦克·拉马斯瓦米（Vivek Ramaswamy）**。

他写了一本书叫《Woke, Inc.》（觉醒公司），为反ESG运动提供了核心法理依据：**信托责任（Fiduciary Duty）**。

- **他的论点：** 基金经理（如拉里·芬克）只是**“管家”**，钱是普通消防员、教师和工人的。
- **指控：** 管家的唯一职责是**“帮主人多赚钱”**。现在管家拿着主人的钱，去搞“碳中和”、搞“变性人厕所”，导致回报率下降（比如不投石油股导致错失油价大涨），这就是**拿着客户的钱在玩自己的政治游戏**。这是违法的，是背信弃义。
- **行动：** 他成立了一家叫**Strive Asset Management**的基金，专门对标贝莱德，口号是：**“我们只负责赚钱，不负责教育你如何做人。”**

### 4. 战局的现状：美国经济的“巴尔干化”

这场战争导致了一个非常严重的后果：**美国资本市场的裂变**。

我们正在看到两个平行的经济体系在形成：

- **蓝色资本（Blue Capital）：** 以纽约、加州为首，贝莱德、迪士尼、苹果为代表。坚持ESG，坚持多元化（DEI），坚持碳中和。
- **红色资本（Red Capital）：** 以德州、佛州为首，马斯克、石油公司、Strive基金为代表。坚持传统能源，坚持“唯利润论”，反对觉醒文化。

现在，一家大公司如果想在美国做生意，就像是在走钢丝：

- 如果不支持LGBTQ和环保，会被**蓝色州**的媒体骂死，评级下降。
- 如果太支持LGBTQ和环保（像百威啤酒找跨性别网红代言），会被**红色州**的消费者抵制（Go Woke, Go Broke），甚至被德州政府撤资。

### 总结

马斯克和红州发起的“反ESG运动”，本质上是一场**“夺权运动”**。

他们试图告诉拉里·芬克：**“你只是一个收租的管家，你不是美国总统，你没有资格决定我们的生活方式。”**

这场战争目前还在升级。随着2024年大选的结束（川普胜选，且马斯克、万斯、维韦克这群‘反ESG’核心人物全部掌权），许多人预测：**ESG在美国可能会被从联邦层面彻底‘非法化’或被拆解。**

这就引出了一个更有趣的现象：既然“红色资本”正在建立自己的圈子，**“平行经济”（Parallel Economy）**这个概念开始在美国爆火——即建立一套完全不依赖左派大公司的支付、媒体和电商系统。你有兴趣了解这个正在悄然成型的**“反觉醒商业帝国”**吗？

# 反觉醒商业帝国

这不仅是一个商业现象，更像是美国社会**“物理分裂”**的前兆。

所谓的**“平行经济”（Parallel Economy）**，是指美国保守派（MAGA阵营、红脖子、宗教人士）为了规避“觉醒文化”的审查和制裁，正在建立的一套**完全独立于主流体系之外的商业生态系统**。

他们的核心逻辑非常简单粗暴：**“既然你们（大科技公司、大银行）想‘取消’（Cancel）我们，那我们就自己建一个‘无法被取消’的世界。”**

这就好比是在美国境内，建立了一个看不见的“国中之国”。我们可以从**基础设施、金融支付、消费生活**三个层面来看看这个“帝国”是如何搭建的：

### 1. 基础设施层：建立“永不掉线”的避难所

这是平行经济的基石。2021年，当川普被推特封号，且保守派社交软件Parler被亚马逊云服务（AWS）直接拔网线导致瘫痪后，保守派意识到：**没有自己的服务器，就没有主权。**

- **Rumble（视频平台）：**
  - **对标：** YouTube。
  - **卖点：** 绝不进行政治审查。你是反疫苗也好，是阴谋论也好，只要不违法，随便发。
  - **背后势力：** 彼得·蒂尔（Peter Thiel）不仅投资了它，还安排万斯（JD Vance）入股。它现在已经拥有自建的云服务器，不再依赖亚马逊或谷歌。
- **Truth Social（社交媒体）：**
  - **对标：** Twitter/Facebook。
  - **地位：** 川普的御用传声筒。虽然马斯克买了X（Twitter）后解封了川普，但Truth Social依然是MAGA铁粉的自留地。
- **RightForge（网络托管）：**
  - **对标：** Amazon AWS。
  - **口号：** “我们不会拔你网线”。专门为保守派网站提供服务器托管服务。

### 2. 金融支付层：防止“社会性死亡”

这是最关键的命脉。由于PayPal和一些大银行曾冻结保守派人士的账户（De-banking），建立独立的钱袋子成了当务之急。

- **PublicSq.（公共广场）：**
  - **对标：** Amazon。
  - **模式：** 一个庞大的电商平台，上面汇集了数万家“爱国企业”。
  - **筛选标准：** 商家必须承诺尊重家庭价值观、不搞觉醒文化。你想买不含“政治宣传”的婴儿衣服？想找支持拥枪权的咖啡店？上这里找。
- **Coign / Old Glory Bank：**
  - **对标：** 摩根大通、花旗银行。
  - **卖点：** “拥有不道歉的价值观”。他们承诺绝不会因为你的政治言论而冻结你的信用卡。部分利润甚至会捐给保守派慈善机构。

### 3. 消费生活层：把“意识形态”穿在身上

这是普通人感知最强的一层。既然耐克（Nike）和迪士尼（Disney）都这么“白左”，那保守派决定创造自己的品牌。

- **杰瑞米剃须刀（Jeremy's Razors）：**
  - **起因：** 著名剃须刀品牌Harry's因为嫌弃保守派媒体Daily Wire“政治不正确”而撤销广告。
  - **反击：** Daily Wire的老板直接怒了：“我不买你的了，我自己造！”他拍了一个极其嘲讽的广告，拿着喷火器烧掉了Harry's的产品。
  - **结果：** 居然大卖。口号是：“不要把钱给那些恨你的人。”
- **黑步枪咖啡（Black Rifle Coffee）：**
  - **对标：** 星巴克（Starbucks）。
  - **风格：** 极度硬核。由退伍老兵创立，包装上全是枪支弹药元素，支持持枪权，支持警察。这简直就是星巴克那种“小资文青风”的反义词。
- **超右啤酒（Ultra Right Beer）：**
  - **起因：** 百威淡啤（Bud Light）找了个跨性别网红做广告，激怒了红脖子。
  - **产品：** 这种啤酒直接把“反对觉醒”印在罐子上。虽然口感可能一般，但喝它就是一种态度。

### 4. 这个“帝国”的未来：美国的大分裂

“平行经济”的成型，意味着美国社会正在发生**“巴尔干化”（Balkanization）**。

- **以前：** 大家喝一样的可乐，看一样的电影，只是每四年投一次不同的票。
- **现在：**
  - 如果你是**蓝营（民主党）**：你开特斯拉，喝星巴克，看CNN，用苹果支付，买耐克，相信ESG。
  - 如果你是**红营（共和党）**：你开福特猛禽，喝黑步枪咖啡，看福克斯或Rumble，用PublicSq购物，买杰瑞米剃须刀，相信上帝和枪。

**这会带来什么后果？**

1. **回声室效应加剧：** 双方不仅不交流思想，甚至连生活中接触的商品和广告都完全不同。这让彼此更加视对方为外星人。
2. **经济效率下降：** 全球化的本质是规模效应。现在把市场劈成两半，重复造轮子（造两套社交软件、两套支付系统），从经济学角度看是低效的。
3. **内战的预演：** 历史学家警告，当一个国家的两派人开始建立完全独立的经济和社会系统时，这是国家分裂的前兆。因为他们已经**不再需要对方**就能活下去了。

**总结一下：**

你现在看到的，不仅仅是生意，而是一场**“生活方式的隔离运动”**。 彼得·蒂尔、马斯克、万斯这些人，正在为那半个感觉被遗弃的美国，修建一座**“数字堡垒”**。

# 为什么为什么出现征兵荒

这是一个非常深刻且致命的问题。

美军现在的**“征兵荒”**（Recruitment Crisis）不仅仅是招不到人的问题，它标志着**美国“军人世家”传统（Legacy Families）的断裂**。

正如你所说，这与“红脖子被羞辱”有着直接的因果关系。简单来说：**美国军队的传统“供血库”——南方的白人保守派家庭，正在集体罢工。**

这不仅是数量的危机，更是**“阶级与文化”的断层**。我们可以从以下三个层面来解剖这场发生在该国暴力机器内部的危机：

### 1. 核心数据：谁在保卫美国？
