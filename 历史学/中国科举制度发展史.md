# 中国科举制度发展史

科举制度（Keju System）是中国历史上通过考试选拔官员的制度。它打破了血缘世袭和门第观念，重塑了中国长达 1300 年的社会结构。

这一制度**始于隋，兴于唐，完善于宋，僵化于明清，废除于清末（1905年）**。

以下是科举制度发展演变的详细全景图：

------

### 一、 诞生期：隋朝 (581–618)

**核心变革：从“拼爹”到“拼才”**

在隋朝之前，官员选拔主要依靠“九品中正制”，实际上被门阀世族垄断（“上品无寒门，下品无势族”）。

- **隋文帝时期：** 废除九品中正制，开始采用分科考试的方式选拔官员。
- **隋炀帝时期：** 正式设立**进士科**，按考试成绩录用人才。标志着科举制度的正式诞生。

> **意义：** 这一变革将选官的权力从世家大族手中收归朝廷，大大加强了中央集权。

------

### 二、 确立与繁荣期：唐朝 (618–907)

**核心变革：制度化与多样化**

唐朝将科举作为选官的主要途径，考试内容和形式逐渐丰富。

- **两大主科：**
  - **进士科（最受重视）：** 考诗赋、策问，难度大，考中者前途最好（“三十老明经，五十少进士”）。
  - **明经科：** 考对儒家经义的记忆和理解，相对容易。
- **武则天的贡献：**
  - 首创**殿试**（皇帝亲自面试）。
  - 开设**武举**，选拔军事人才。
- **特点：** 此时考试尚未完全密封试卷，考官的主观意愿和考生的名声（行卷）对结果有很大影响。

------

### 三、 完善与改革期：宋朝 (960–1279)

**核心变革：追求极致的公平**

宋朝重文轻武，科举成为社会流动的最主要通道。宋朝对考试程序进行了严格的改革，以防作弊。

- **锁院制度：** 考官在考试期间不得回家或见客。
- **糊名（弥封）：** 将试卷上的考生姓名籍贯密封，考官不知道在批谁的卷子。
- **誊录：** 雇专人将考生的试卷重新抄写一遍，防止考官辨认字迹。
- **扩大录取：** 大幅度增加录取名额，许多平民子弟通过科举成为宰相（如范仲淹、欧阳修）。

> **关键点：** 宋朝确立了科举作为选拔“士大夫”阶层的绝对核心地位，形成了“万般皆下品，唯有读书高”的社会风气。

------

### 四、 僵化期：明清 (1368–1905)

**核心变革：八股取士与三级考试体系**

到了明清，为了统一思想和便于评判，科举制度变得极度标准化，但也因此失去了活力。

#### 1. 考试内容的束缚

- **指定教材：** 题目必须出自**《四书》、《五经》**。
- **标准答案：** 解释必须以朱熹的《四书集注》为准，不得自由发挥。
- **文体限制：** 必须使用**“八股文”**（排比对偶，格式严格）。这极大地禁锢了读书人的思想。

#### 2. 严密的晋升阶梯

明清形成了最为人们熟知的“三级考试”金字塔结构：

| **级别**   | **考试名称** | **地点** | **考中者称谓**  | **第一名** | **俗称/特权**            |
| ---------- | ------------ | -------- | --------------- | ---------- | ------------------------ |
| **预备**   | 童试         | 县/府    | **秀才** (生员) | 案首       | 见官不跪，免除徭役       |
| **第一级** | 乡试 (秋闱)  | 省城     | **举人**        | 解元       | 有做官资格，社会地位大增 |
| **第二级** | 会试 (春闱)  | 京城     | **贡士**        | 会元       | 获得殿试资格             |
| **第三级** | 殿试         | 皇宫     | **进士**        | **状元**   | 天子门生，进入官僚核心   |

- **殿试三甲：**
  - 一甲三人：状元、榜眼、探花（赐进士及第）。
  - 二甲若干：赐进士出身。
  - 三甲若干：赐同进士出身。

------

### 五、 终结：清末 (1905)

**核心变革：废科举，兴学堂**

19世纪末，面对列强入侵，传统的科举制度选拔的人才已无法应对现代化的挑战（不懂科学、技术、外交）。

- **1898年：** 戊戌变法曾尝试改革科举，增加经济特科，废八股改策论。
- **1905年：** 在袁世凯、张之洞等大臣的奏请下，光绪皇帝下诏**正式废除科举制度**，推广新式学堂。

至此，延续了 1300 年的科举制度彻底退出历史舞台。

------

### 总结：科举制的历史功过

为了便于您理解，我们可以从正反两方面来看：

- **正面价值（Pros）：**
  1. **公平性：** 打破了贵族垄断，提供了“朝为田舍郎，暮登天子堂”的阶层流动机会。
  2. **稳定性：** 将全天下的精英吸纳到体制内，维持了国家的统一和稳定。
  3. **重学风气：** 普及了儒家文化，提高了官员的文化素质。
- **负面影响（Cons）：**
  1. **思想禁锢：** 尤其是明清八股文，扼杀了创造力和独立思考。
  2. **忽视实用：** 考试内容脱离实际，导致官员往往不懂科技和经济，这也是中国近代落后的原因之一。
